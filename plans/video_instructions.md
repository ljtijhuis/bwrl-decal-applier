How to Add Decals to Your iRacing Paint File — Overview:
- Two main methods: using GIMP (desktop) or Photopea (online)
- Final step: upload decal and spec map to TradingPaints (or use TradingPaints Pro decal layer if subscribed)

Prerequisites:
- Decal files downloaded (example: Corvette decal) — from provided Google Drive link
- Have run the paint in TradingPaints beforehand so TradingPaints downloads the TGA into your iRacing paints folder
- Know your iRacing Customer ID (found via your account, top-right)

Folder / File Locations:
- iRacing paints folder path: Documents/iRacing/paint/[car folder]
- Decal download location (example): Downloads/Decal Upload
- Decal file types:
  - Decal layer: PNG or TGA (from decal download)
  - Spec map: MIP file, named car_spec_[yourID].map (same number as TGA)
- Export naming convention example: car_[your iRacing ID].tga (or car_spec_[yourID].map)

Method 1 — GIMP (free desktop app):
1. Download and install GIMP from GIMP.org
2. Open your paint TGA:
   - Find your paint in Documents/iRacing/paint/[car], drag-and-drop into GIMP
3. Import decal:
   - Open decal folder, drag-and-drop appropriate car decal (e.g., Corvette Z06) into GIMP
   - If prompted about color profile import, click "Keep"
   - Decal imports as a layer on top of car layer
4. Export edited file:
   - File → Export As (choose TGA)
   - Name file as car_[yourID].tga and choose destination (example: Downloads/Decal Upload)
   - Click Export when prompted
5. Locate exported TGA and corresponding spec map (.mip) for upload

Method 2 — Photopea (free online editor):
1. Open photopea.com
2. Open your paint TGA:
   - Drag-and-drop the TGA into Photopea
3. Import decal PNG/TGA:
   - Drag-and-drop the decal PNG/TGA into Photopea (it becomes a layer)
4. Export edited file:
   - File → Export As → TGA (set resolution example 2048x2048 if applicable)
   - Save to downloads folder
5. Locate exported TGA and corresponding spec map (.mip) for upload

Uploading to TradingPaints:
1. Go to tradingpaints.com → Upload
2. Select "Select a paint file" → choose the exported TGA (Downloads/Decal Upload)
3. Also upload the spec map:
   - Select spec map file (car_spec_[yourID].map or .mip)
   - Upload spec map to ensure shiny/matte elements display correctly
4. Select correct vehicle (example: Corvette Z06 GT3R)
5. Choose upload destination (example: "Upload to my paints")
6. Complete upload — decal will show for others and yourself when running TradingPaints

TradingPaints Pro option:
- If you have TradingPaints Pro subscription, you can add a decal layer directly in TradingPaints
- Use the appropriate Pro decal file (e.g., Corvette Z06 GT3R Pro decal or class-specific decal)
- No external editor required when using the TradingPaints Pro decal layer

Notes / Tips:
- Ensure TradingPaints has already downloaded the original TGA to your paints folder before editing
- Keep track of your iRacing Customer ID for correct file naming
- Upload both the decal TGA and its corresponding spec map for proper appearance
